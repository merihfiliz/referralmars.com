# ReferralMars – Kripto Referral WordPress Teması

Kripto borsalarının referral kodlarını, bonuslarını ve incelemelerini paylaşmak için tasarlanmış modern dark tema.

## Özellikler
- Her yazı için özel meta box (borsa adı, referral kodu, link, bonus)
- Single post'ta şık referral kutusu + tek tık kopyalama
- Neon turuncu gradient + Orbitron/Inter fontları
- Hero bölüm, post kartları, sidebar, footer widget desteği
- Responsive mobil menü

## Kurulum
1. Bu repoyu zip olarak indirin  
2. WordPress → Görünüm → Temalar → Yeni Ekle → Tema Yükle  
3. Zip'i yükleyin ve etkinleştirin  
4. Yazı eklerken sağ taraftaki "Referral / Borsa Bilgileri" kutusunu doldurun
